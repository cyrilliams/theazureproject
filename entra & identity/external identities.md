# Inviting an External User

In this doc, we'll invite a user from a domain that is not ours. (A G-Mail one in this case)

## Invite External User

In Entra ID, we'll go to **Entra ID > Users > New User > Invite external user**

<img width="343" height="75" alt="image" src="https://github.com/user-attachments/assets/3fb4d18b-991d-4b6c-8cca-30e766e1f70c" />

Next, we'll fill out the info for the external user:

<img width="2535" height="778" alt="image" src="https://github.com/user-attachments/assets/869e6840-f6a8-4bab-93c3-1432d4a81584" />

Fill out the necessary properties:

<img width="1148" height="946" alt="image" src="https://github.com/user-attachments/assets/aa66da15-bb7b-4d88-aac1-3f1831b52bc2" />

Add groups or roles if needed:

<img width="982" height="342" alt="image" src="https://github.com/user-attachments/assets/ece328a1-6f78-4260-8d78-72e224714b1b" />

Now, review and invite:

<img width="867" height="1175" alt="image" src="https://github.com/user-attachments/assets/bc91ad4f-5150-43c9-a764-b933e0733d6d" />

## Invite

Now, if we check that mailbox, we can see our invite here:

<img width="1621" height="1139" alt="image" src="https://github.com/user-attachments/assets/374b5f67-816b-4e26-9697-4eee5491ffdd" />
