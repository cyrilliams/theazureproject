# Set Up a Private DNS Zone

There are two types of DNS Zones: Public & Private

We'll create a Private DNS Zone, which is resolvable only within our Virtual Networks. These cannot be resolved over the internet.

## Create DNS Zone

We'll search for 'Private DNS Zones'.

<img width="511" height="77" alt="image" src="https://github.com/user-attachments/assets/ddf794d1-88fe-442d-b91b-08f3c2fa6eb0" />

Click 'Create'

We'll enter what we want our Private Zone to resolve to and then Create:

<img width="968" height="673" alt="image" src="https://github.com/user-attachments/assets/821eb0b0-4c3e-4b7f-a23f-b05f088a3b38" />
