# Create an SSH Key Pair

In this doc, we'll create an SSH key pair named ``devops-kp`` with the type being ``rsa``.

We'll go to **SSH Keys > Create**

Name the key pair and select ``RSA SSH Format``

<img width="1056" height="896" alt="image" src="https://github.com/user-attachments/assets/d9cc2d9a-0839-46a3-b2da-b9b9840c16f3" />


Review + Create.

We get the option to download our Private key.
